# HSE Project Template

## Описание

Данный репозиторий представляет собой шаблон C++-проекта с разделением исходного кода, заголовочных файлов, тестов и сторонних зависимостей.  
Проект использует систему сборки **CMake**, а также поддерживает сборку через **Makefile**.  
Для написания и запуска модульных тестов используется библиотека **Google Test**.

---

## Структура проекта

```text
hse-project-template/
├── include/
├── src/
├── tests/
├── third_party/
│ ├── boost/
│ └── gtest/
├── CMakeLists.txt
├── Makefile
└── main.cpp
```

### Описание каталогов

**include/**  
Каталог с заголовочными файлами проекта (`.h`, `.hpp`).  
Здесь размещаются объявления функций, классов и интерфейсов, которые используются в исходном коде.

**src/**  
Каталог с реализацией основной логики программы.  
Содержит исходные файлы (`.cpp`), реализующие функциональность, объявленную в заголовочных файлах.

**tests/**  
Каталог для модульных тестов.  
Тесты проверяют корректность работы отдельных компонентов программы и написаны с использованием библиотеки **Google Test**.

**third_party/**  
Каталог сторонних библиотек, используемых в проекте.

- **boost/** — библиотека Boost, расширяющая возможности стандартной библиотеки C++.
- **gtest/** — библиотека Google Test, используемая для написания и запуска тестов.

---

## Основные файлы

**CMakeLists.txt**  
Главный конфигурационный файл системы сборки **CMake**.  
Определяет цели сборки, зависимости проекта и правила компиляции.

**Makefile**  
Файл для сборки проекта с помощью утилиты **make**.

**main.cpp**  
Точка входа в программу.  
Содержит функцию `main()`, с которой начинается выполнение приложения.
